工作：需要一个容器的id不能是数组， 一个可移动的对象id， 一个课关闭的对象id， 窗口的坐标， 是否可以移出浏览器窗口， 4个过渡效果，（注意：如果过渡类型是inflate zoom： 1 or 0 从水平改变尺寸，1 or 0 从垂直改变尺寸）， 动画时间， 是否中断
